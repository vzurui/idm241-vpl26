Alpha Build

"Add to Library"

TRIGGER

Save Library - Default

  • Hover ON
    • 1 second delay
  • Hover OFF
  • Mouse Down
  • Mouse Up (Click ON)

Save Library - Active

  • Hover ON
    • 1 second delay
  • Hover OFF
  • Mouse Down
  • Mouse Up (Click OFF)

RULES

Save Library - Default

  • Hover ON - Cursor changes to finger, scale changes to 110%
    • 1 second delay "Save to Your Library" appears as tagline
  • Hover OFF - Cursor changes back to pointer
  • Mouse Down - Icon opacity changes to 50%, scale changes to 100%
  • Mouse Up (Click ON) - Icon outline fills in with green, changes to check mark, confetti pops out
  • Database record flag set to "TRUE"

Save Library - Active

  • Hover ON - Cursor changes to finger, scale changes to 110%
    • 1 second delay "Remove from Your Library" appears as tagline
  • Hover OFF - Cursor changes back to pointer
  • Mouse Down - Icon opacity changes to 50%, scale changes to 100%
  • Mouse Up (Click OFF) - Icon outline goes back to default outline
  • Database record flag set to "FALSE"

Ctrl

SZA

• 2017 • 14 songs, 49 mins 6 sec

SZA's Ctrl, released on June 9, 2017, redefined modern R&B with its raw emotion and relatable storytelling. Blending themes of love, heartbreak, self-doubt, and empowerment, it features standout tracks like “Love Galore” and “The Weekend” with collaborations from Travis Scott and Kendrick Lamar. Praised for its honest lyrics and innovative sound, Ctrl propelled SZA to stardom as a trailblazing voice in contemporary music.

RULES

Save Library - Default

  • Hover ON - Cursor changes to finger, opacity changes to 50%
    • 1 second delay "Save to Your Library" appears as tagline
  • Hover OFF - Cursor changes back to pointer
    • Tagline disappears
  • Mouse Down - Heart icon opacity changes to 50%
  • Mouse Up (Click ON) - Heart icon outline fills in with green, confetti pops out
  • Database record flag set to "TRUE"

Save Library - Active

  • Hover ON - Cursor changes to finger
    • 1 second delay "Remove from Your Library" appears as tagline
  • Hover OFF - Cursor changes back to pointer
    • Tagline disappears
  • Mouse Down - Heart icon opacity changes to 50%
  • Mouse Up (Click OFF) - Heart fades back to original outline
  • Database record flag set to "FALSE"

More Options - Default

  • Hover ON - Cursor changes to finger, dots move in Y position up and down (bounce)
    • 1 second delay "More Options" appears as tagline
  • Hover OFF - Cursor changes back to pointer
    • Tagline disappears
  • Mouse Down - Options icon opacity changes to 50%
  • Mouse Up (Click ON) - Dots change Y position, option mode pops up

More Options - Active

  • Hover ON - Cursor changes to finger, dots move in Y position up and down (bounce)
    • 1 second delay "Less Options" appears as tagline
  • Hover OFF - Cursor changes back to pointer

  • Mouse Up (Click OFF) - Dots change Y position, option mode pops up

Shuffle Button - Default

  • Hover ON - Cursor changes to finger
    • 1 second delay "Enable Shuffle" appears as tagline
  • Hover OFF - Cursor changes back to pointer
    • Tagline disappears
  • Mouse Down - Shuffle icon opacity changes to 50%
  • Mouse Up (Click ON) - Shuffle icon fades into green, both arrows do a Y reflection to switch positions (flip)

Shuffle Button - Active

  • Mouse Down - Shuffle icon opacity changes to 50%
  • Mouse Up (Click OFF) - Shuffle icon fades back into default color, both arrows do a Y reflection to switch positions (flip)

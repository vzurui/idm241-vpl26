RULES

Save Library - Default

  • Hover ON - Cursor changes to finger, scale changes to 110%
    • 1 second delay "Save to Your Library" appears as tagline
  • Hover OFF - Cursor changes back to pointer
    • Tagline disappears
  • Mouse Down - Heart icon opacity changes to 50%, scale changes to 100%
  • Mouse Up (Click ON) - Heart icon outline fills in with green, changes to check mark, confetti pops out
  • Database record flag set to "TRUE"

Save Library - Active

  • Hover ON - Cursor changes to finger, scale changes to 110%
    • 1 second delay "Remove from Your Library" appears as tagline
  • Hover OFF - Cursor changes back to pointer
    • Tagline disappears
  • Mouse Down - Heart icon opacity changes to 50%, scale changes to 100%
  • Mouse Up (Click OFF) - Heart icon outline pulses twice like a heartbeat, then goes back to default outline
  • Database record flag set to "FALSE"

More Options - Default

  • Hover ON - Cursor changes to finger, dots move in Y position up and down, scale changes to 110%
    • 1 second delay "More Options" appears as tagline
  • Hover OFF - Cursor changes back to point
    • Tagline disappears
  • Mouse Down - Dots icon opacity changes to 50%, scale changes to 100%
  • Mouse Up (Click ON) - Dots change Y position, option mode pops up

More Options - Active

  • Hover ON - Cursor changes to finger, dots move in Y position up and down, scale changes to 110%
    • 1 second delay "Less Options" appears as tagline
  • Hover OFF - Cursor changes back to point
    • Tagline disappears
  • Mouse Down - Dots icon opacity changes to 50%, scale changes to 100%
  • Mouse Up (Click OFF) - Dots change Y position, option mode pops up

Shuffle Button - Default

  • Hover ON - Cursor changes to finger, scale changes to 110%
    • 1 second delay "Enable Shuffle" appears as tagline
  • Hover OFF - Cursor changes back to pointer
    • Tagline disappears
  • Mouse Down -Shuffle icon opacity changes to 50%, scale changes to 100%
  • Mouse Up (Click ON) - Shuffle icon fades into green, both arrows do a Y translation to switch position (up and down)

Shuffle Button - Active

  • Hover ON - Cursor changes to pointer, scale changes to 110%
    • 1 second delay "Disable Shuffle" appears as tagline
  • Hover OFF - Cursor changes back to pointer
    • Tagline disappears
  • Mouse Down -Shuffle icon opacity changes to 50%, scale changes to 100%
  • Mouse Up (Click OFF) - Shuffle icon fades back into default color, both arrows do a Y translation to switch position (up and down)
